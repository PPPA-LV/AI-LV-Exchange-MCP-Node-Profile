## Why two licences

A trust profile is only useful if anyone can implement it — including commercially, including a competitor of any of its authors. Section 16 of the profile commits the Exchange to vendor neutrality and to implementation-neutral test criteria. A licence that restricted commercial use would contradict that commitment, so neither licence does.

The schemas and code are MIT so that they can be copied directly into an implementation with no obligation beyond attribution of the notice. The specification text is CC BY 4.0 so that it can be quoted, translated, adapted and profiled downstream, provided the source is credited.
